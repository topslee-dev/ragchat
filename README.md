## 🎯 프로젝트 개요

전자금융거래법이라는 복잡한 법률 문서를 AI가 이해하고, 사용자의 질문에 **정확한 조항과 함께** 답변하는 챗봇입니다.

**간단히 말해:**
- 📄 **문제**: 법률 문서는 복잡하고 찾기 어렵다
- 💡 **해결**: AI가 자동으로 관련 조항을 찾아서 설명해준다
- ✅ **결과**: "전자금융거래란 무엇인가요?" → 2초 만에 제2조 원문 + 설명

---

## 🚀 이 프로젝트로 증명한 개발 능력

| 능력 | 설명 | 구현 사항 |
|------|------|---------|
| **1. 시스템 설계** | 요구사항을 분석해서 기술 선택 | PDF → AI 답변까지 전체 파이프라인 설계 |
| **2. 데이터 처리** | 비정형 데이터를 기계가 이해할 수 있는 형태로 변환 | 500 페이지 법률 문서를 효율적으로 처리 |
| **3. 오픈소스 활용** | 필요한 기술을 찾아서 조합 | LangChain, Ollama, Chroma 등 6개 라이브러리 통합 |
| **4. 로컬 AI 운영** | 클라우드 없이 로컬에서 AI 모델 실행 | Mistral 7B 모델을 4GB 환경에서 구동 |
| **5. 기본 NLP 이해** | 텍스트를 컴퓨터가 이해하는 형태로 변환 | 문서를 벡터로 변환하고 유사도 검색 |
| **6. 품질 관리** | 답변이 정확한지 검증 | 법률 조항 번호 인용으로 신뢰도 확보 |
| **7. 최적화** | 제한된 자원으로 최대 성능 발휘 | 4GB VRAM에서 3초 이내 답변 생성 |

## 📋 빠른 시작

### 1️⃣ 설치 (3단계)

```bash
# 1. 저장소 클론
git clone https://github.com/yourusername/RagChat.git
cd RagChat

# 2. Python 환경 설정 (처음 1번)
python -m venv .venv
.venv\Scripts\activate  # Windows

# 3. 라이브러리 설치
pip install -r requirements.txt
```

### 2️⃣ AI 모델 준비 (처음 1번)

```bash
# 1. Ollama 다운로드 및 실행
# → https://ollama.ai 에서 설치

# 2. 새 터미널에서 서버 시작
ollama serve

# 3. 다른 터미널에서 모델 다운로드 (약 5분)
ollama pull mistral:latest

# 4. 벡터 데이터베이스 생성 (약 5분)
python src/vectorstore.py
```

### 3️⃣ 챗봇 실행

```bash
python main.py
```

**실행 결과:**
```
질문: 전자금융거래란 무엇인가요?

답변: 제2조에 따르면, '전자금융거래'는 금융거래의 당사자가 
전자적 수단을 이용하여 행하는 금융거래를 의미합니다.

예: 인터넷 뱅킹, 모바일 뱅킹, 전자지급수단 등

(응답 시간: 2.34초)
```

## 🔧 작동 원리 (간단 버전)

```
1️⃣ 문서 학습 (처음 1번)
   PDF 파일 읽기 → 작은 조각으로 분할 → 의미를 벡터로 변환 → 저장

2️⃣ 사용자 질문 시 (매번)
   "전자금융거래란?" 
       ↓
   동일한 방식으로 벡터 변환
       ↓
   저장된 벡터 중 가장 비슷한 문서 찾기
       ↓
   "관련 조항: 제2조입니다" 라고 AI가 답변
```
